# 🔧 快速修复401错误

## 问题
```
POST http://localhost:3001/api/ai/adapt-platform 401 (Unauthorized)
wechat 适配失败: Error: 无效的认证令牌
```

## 原因
刷新页面后，localStorage的token可能丢失或过期

## ✅ 解决方法（30秒）

### 步骤1：退出登录
在页面右上角点击用户菜单 → 退出登录

### 步骤2：重新登录
```
邮箱: test@test.com
密码: 123456
```

### 步骤3：测试
- 打开或创建一篇文章
- 点击 🚀 按钮
- 选择平台
- 开始适配
- ✅ 应该能看到Tab切换界面了

---

## 🔍 为什么会这样？

1. Vite热更新有时会清空localStorage
2. 代码改动后，token存储机制变化
3. 需要重新获取新的token

## 💡 如果还是401

打开浏览器控制台（F12），输入：
```javascript
localStorage.getItem('auth_token')
```

如果返回 `null`，说明确实没有token，重新登录即可。
