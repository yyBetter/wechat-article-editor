## 🔧 自定义配置

### 关闭引导卡片
用户首次看到引导后，点击"知道了"或关闭按钮，会在localStorage中记录：
```javascript
localStorage.setItem('smart_paste_guide_seen', 'true')
```

### 重新显示引导
如果需要重新显示引导，清除localStorage：
```javascript
localStorage.removeItem('smart_paste_guide_seen')
```

或者在浏览器控制台执行：
```javascript
localStorage.clear()
```
