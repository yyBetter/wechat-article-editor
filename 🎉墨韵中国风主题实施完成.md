- **深宣纸** `#f0ebe3` - 次背景

#### 金色系（装饰色）
- **金色** `#d4af37` - 边框、分隔线
- **浅金** `#e6c975` - 辅助装饰

### 🖼️ 2. 宣纸纹理系统

**全局可复用类**: `.xuan-paper-texture`

**视觉效果**：
- ✅ 细腻的纸张纤维交叉纹理
- ✅ 模拟纸张瑕疵的随机斑点
- ✅ 温润的质感体验

**应用区域**：
- Hero Section
- 功能展示区
- 平台适配区
- AI 数字员工介绍区

### 🖋️ 3. 水墨渐变元素

**4 种渐变**：
```css
ink-gradient      /* 墨黑渐变 - 主按钮、标题 */
cinnabar-gradient /* 朱砂渐变 - 徽章、强调 */
gold-gradient     /* 金色渐变 - 装饰元素 */
paper-gradient    /* 宣纸渐变 - 背景过渡 */
```

**水墨晕染效果**：
- 底部自动添加渐变晕染
- 模拟水墨画的自然过渡

### 🔖 4. 中国风组件

#### 印章样式徽章
```css
.seal-badge
- 朱砂红边框 + 内阴影
- 轻微旋转(-2deg)  
- 菱形装饰符号
```
**使用场景**: "独家功能"、"核心成员"等标签

#### 竹简分隔线
```css
.bamboo-divider
- 金色渐变线条
- 居中菱形装饰
- 用于区块分隔
```
**使用场景**: 内容区块之间的优雅分隔

### 📐 5. 设计系统

#### 字体系统
- **标题**: 思源宋体 / 楷体（衬线字体）
- **正文**: 苹方 / 微软雅黑（无衬线字体）

#### 阴影系统（5 级）
- `shadow-sm` - 轻微阴影
- `shadow-md` - 中等阴影
- `shadow-lg` - 明显阴影
- `shadow-xl` - 强阴影
- `shadow-ink` - 水墨阴影（特殊强调）

#### 圆角系统
- `4px` - 小元素（标签、徽章）
- `8px` - 中等元素（按钮、输入框）
- `12px` - 大元素（卡片）
- `16px` - 特大元素（模态框）

#### 动画系统
- `transition` - 标准过渡 (0.3s)
- `transition-ink` - 水墨过渡 (0.4s, 更柔和)
