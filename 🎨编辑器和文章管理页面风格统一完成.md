## ✨ 解决方案

### 墨韵中国风设计系统

#### 1. 颜色系统

```css
:root {
  /* 主色系 - 水墨色 */
  --ink-black: #2c3e50;       /* 墨黑 - 主文字 */
  --ink-grey: #5a6c7d;        /* 墨灰 - 次要文字 */
  
  /* 强调色 - 朱砂红 */
  --cinnabar-red: #d32f2f;    /* 朱砂红 - 警告/悬停 */
  
  /* 背景色 - 宣纸色 */
  --xuan-paper: #f9f6f0;      /* 浅宣纸 - 主背景 */
  --xuan-paper-dark: #f0ebe3; /* 深宣纸 - 卡片背景 */
  
  /* 装饰色 - 金色 */
  --gold-accent: #d4af37;     /* 金色 - 边框/强调 */
  --gold-light: #e6c975;      /* 淡金 - 渐变/淡化 */
  
  /* 文字色 */
  --color-text: #2c3e50;      /* 主文字 */
  --color-text-light: #5a6c7d;/* 次要文字 */
  
  /* 边框色 */
  --color-border: #d4c5b0;    /* 默认边框 */
}
```

#### 2. 阴影系统

```css
:root {
  --shadow-sm: 0 2px 4px rgba(44, 62, 80, 0.08);
  --shadow-md: 0 4px 12px rgba(44, 62, 80, 0.12);
  --shadow-lg: 0 8px 24px rgba(44, 62, 80, 0.16);
  --shadow-ink: 0 4px 16px rgba(44, 62, 80, 0.25);  /* 水墨阴影 */
}
```

#### 3. 圆角系统

```css
:root {
  --radius-sm: 4px;   /* 小元素：标签、徽章 */
  --radius-md: 8px;   /* 中元素：按钮、输入框 */
  --radius-lg: 12px;  /* 大元素：卡片 */
}
```

#### 4. 动画系统

```css
:root {
  --transition-ink: all 0.4s cubic-bezier(0.25, 0.46, 0.45, 0.94);
}

/* 统一的悬停效果 */
.button:hover {
  transform: translateY(-2px);  /* 上浮2px */
  box-shadow: var(--shadow-md);
  border-color: var(--gold-accent);  /* 金色高亮 */
}
```

#### 5. 字体系统

```css
:root {
  --font-family-title: 'Source Han Serif CN', 'Noto Serif SC', 'KaiTi', serif, -apple-system;
}

/* 标题使用书法字体 */
h1, h2, h3 {
  font-family: var(--font-family-title);
  font-weight: 700;
  color: var(--ink-black);
  letter-spacing: 1px;
}
```

## 💡 设计思考

### 1. 为什么要统一样式？

#### 用户体验角度
- **认知一致性**：用户在不同页面看到相同的设计语言，学习成本更低
- **品牌记忆**：重复出现的设计元素（金色边框、墨黑渐变）强化品牌印象
- **操作连贯**：统一的交互反馈（上浮2px）让操作更流畅

#### 品牌建设角度
- **差异化特色**：墨韵中国风是我们与国外产品的核心差异
- **文化自信**：不再模仿Material Design，有自己的设计语言
