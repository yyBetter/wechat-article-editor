## 🚀 使用方法

### Step 1: 启动服务

```bash
# 启动后端
cd server
npm run dev

# 启动前端
cd ..
npm run dev
```

### Step 2: 打开编辑器

1. 登录系统
2. 进入编辑器页面
3. 查看工具栏

### Step 3: 使用AI工作室

1. **找到按钮**：工具栏中的 ✨ 按钮（紫色渐变，带闪烁效果）
2. **点击打开**：打开AI写作工作室模态框
3. **选择风格**：点击你喜欢的写作风格卡片
4. **输入内容**：在左侧输入框粘贴或输入要改写的内容
5. **开始改写**：点击"🚀 开始改写"按钮
6. **查看结果**：右侧显示改写后的内容
7. **应用内容**：点击"✅ 应用到编辑器"
